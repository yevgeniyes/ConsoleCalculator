# ConsoleCalculator
Performing basic math operations, operations with local files and some test operations with Facebook API (used base OOP principles, fabric and template patterns)

Available operations for calculator: sum (+), sub (-), mult (*), div (/). Example command: sum 3 2.54 18

Available operations for files: create, delete, copy, move. Example command: create C:\New\SomeFile.txt

Facebook commands allows to work with test users: fb list - return test users list, fb username - create test user and return user id
